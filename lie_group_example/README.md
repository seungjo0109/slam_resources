# Lie Group Example
* Lie Group Example은 Lie group, Lie algebra를 다루는 저장소입니다.
* Lie group, Lie algebra 예제는 Sophus 라이브러리를 활용합니다.

### Examples
* lie_group_example

## 1. Pre-requisite
* [Sophus](https://github.com/strasdat/Sophus)
* Eigen3
* fmt

## 2. Compilation
```
mkdir build && cd build
cmake ..
make
```

## 3. Run
```
./lie_group_example
```
